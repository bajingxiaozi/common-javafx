# common-javafx

> Java-fx工具类

已经发布到了使用github搭建的maven仓库上

## 使用步骤

1. 添加服务器地址

```
repositories {
    mavenCentral()
    ...
    maven { url = "https://raw.githubusercontent.com/bajingxiaozi/maven-repo/master/" }
}
```

1. 添加依赖

```
dependencies {
    ...
    compile 'com.github.bajingxiaozi:common-javafx:1.0-SNAPSHOT'
}
```
